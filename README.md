### **사용자 Flow (Usecase)**

- 맛집을 등록할 수 있다
- 맛집을 수정할 수 있다
- 맛집을 삭제할 수 있다
- 맛집에 리뷰를 작성할 수 있다
- 맛집에 작성한 리뷰를 삭제할 수 있다
- 맛집에 작성된 리뷰와 평균별점을 확인할 수 있다

### **데이터**

- 맛집
    - 이름 (String)
    - 주소 (String)
    - N개의 메뉴
        - 이름 (String)
        - 가격 (Number)
    - N개의 리뷰
        - 본문 (String)
        - 별점 (Number)
